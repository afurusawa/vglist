About Website

This is a website made for users to keep track of the games people play across all consoles. There have been several similar websites that do this but they're all a clutter, and keeping track of games isn't their main focus sometimes. Features to include are minor social aspects such as adding friends and viewing compatibility between friends based on personal game ratings and games played

Security Details

To ensure basic security, all passwords have been encrypted using a salted hash for users who didn't sign up using facebook.

In case this project is unsustainable, the games database will be dumped into a file, and users will be able to download a csv of their personal rating info and other stats.

Compatibility algorithm

Thinking of a good algorithm has been a challenge. First and foremost, the key factor is matching ratings, which is comparing the rating of both parties and generating a percentage based on closeness

Also, rating a game well and personally selecting a game as a favorite can be two different things. Favorite games can be compared as well and be taken into account in generating the compatibility rating.

Other key factors to consider is platform, genre, volume of games played. Not sure how this will be included.
